# SKM - Sistem Keuangan Madrasah

Aplikasi desktop Java Swing untuk pengelolaan keuangan madrasah: dashboard KPI, data master, pemasukan, pengeluaran, tabungan siswa, laporan PDF/Excel, pengaturan, role, dan audit log.

## Menjalankan
1. Import `database/kas_sekolah.sql` ke MySQL 8.
2. Sesuaikan `config.properties`.
3. Jalankan `mvn clean compile exec:java`.
